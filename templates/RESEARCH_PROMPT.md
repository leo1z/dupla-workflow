Help me structure and refine a product idea into a high-quality Idea Draft.

---

## Phase 1 — Discovery (MANDATORY)

Before generating anything:

Ask me the minimum set of questions needed to clearly understand the idea.

Rules:
- Max 6–8 questions
- Focus ONLY on:
  - Problem
  - User
  - Core idea
  - Core flow
  - Constraints
- Avoid unnecessary questions

Wait for answers.

---

## Phase 2 — Idea Draft Generation

Generate the Idea Draft using the template below.

Rules:
- Max 600 tokens
- No technical decisions (no stack, no architecture)
- Use bullet points
- Be concise and structured
- Optimize for future Claude usage (clarity > explanation)

---

## Phase 3 — Clarity Check

- Complete Clarity Check
- If ANY = NO → auto-refine
- Repeat until all YES

---

## Output Rules

- Return ONLY final Idea Draft
- No explanations
- No commentary
